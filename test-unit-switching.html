Unit Switching Test

This test verifies that unit switching works correctly on the BMI calculator.

Test Instructions:

  1. Go to BMI Calculator
  2. Try switching height units from ft/in to cm and back
  3. Try switching weight units from lbs to kg and back
  4. Verify that:
    • The input fields change appropriately
    • Values are converted correctly
    • BMI calculation updates

Expected Behavior:

  • Height switching: When switching from ft/in to cm, the separate feet/inches inputs should hide and a single cm input should appear
  • Weight switching: When switching from lbs to kg, the lbs input should hide and kg input should appear
  • Value conversion: Values should be automatically converted between units
  • BMI update: BMI result should update immediately after unit change

Debug Console:

Open browser developer tools (F12) and check the console for debug messages like:

  • "Unit change event received: {attribute: 'unit', value: 'metric'}"
  • "Label text: height New value: metric"
  • "Updating height unit to: metric"
